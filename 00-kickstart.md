# Meeting 0: Первые шаги (Kickstart) 🌱

> **Опциональная ознакомительная встреча.** Если ты впервые в мире AI-coding и прыгать сразу в Claude Code страшно — начни отсюда. Никаких установок и домашки. Только карта местности.

---

## ⚡ Активация: Твоя идея-призрак

Вспомни проект, приложение или автоматизацию, которую ты давно хотел сделать, но **откладывал**, потому что «это слишком сложно технически», «нужно нанимать программиста» или «у меня нет полгода на изучение Python».

**Что это за идея?** Опиши её парой слов.

---

## 🔄 Рефлексия: Цена входа

В старом мире (до 2024 года) для реализации этой идеи тебе бы понадобилось:
1.  **3–6 месяцев** на изучение основ программирования.
2.  **Или $2000–$5000** на оплату фрилансера.
3.  **Десятки часов** на отладку багов, которые ты даже не понимаешь.

**Чувствуешь, как это барьер убивает мотивацию?** Vibe coding придуман, чтобы убрать этот барьер.

---

## 🧠 Концепция: Что такое vibe coding?

Простыми словами: **ты говоришь — AI пишет код**.

### Разница подходов

| Роль | Старая разработка (Software 1.0/2.0) | Vibe coding (Software 3.0) |
| --- | --- | --- |
| **Кто решает ЧТО делать** | Product Manager | **ТЫ** |
| **Кто пишет код** | Программист (3-5 лет опыта) | **AI-агент** |
| **Кто проверяет результат** | Тестировщик | **ТЫ** |
| **Кто запускает в мир** | DevOps-инженер | **ТЫ (с помощью AI)** |

**Твой новый стек:** Ты — дирижёр. AI — оркестр. Тебе не нужно уметь играть на каждой скрипке, тебе нужно знать партитуру.

---

## 🗺️ Карта AI-инструментов

Чтобы не утонуть в море хайпа, запомни 4 квадранта:

| Квадрант | Что делает | Примеры |
| --- | --- | --- |
| 🟦 **Zero-code builders** | Промпт → готовое приложение за минуты | Lovable, Bolt, Replit |
| 🟪 **AI dev platforms** | Глубокая работа с кодом, файлами, проектом | **Claude Code**, Codex |
| 🟨 **Design-to-code** | Дизайн → код | Figma Make, Stitch |
| 🟩 **AI coding assistants** | Помощники внутри IDE (подсказки) | Cursor, Windsurf |

**Claude Code** (наш фокус) — это самый мощный инструмент. Он не просто подсказывает код, он **действует** в твоей файловой системе и терминале.

---

## 📚 Минимальный словарь

| Термин | Суть |
| --- | --- |
| **LLM** | "Мозг" AI (Claude, GPT). |
| **Промпт** | Твое задание для AI. |
| **Агент** | LLM + инструменты (может сам запускать команды). |
| **Frontend/Backend** | Лицо приложения / Его скрытая логика. |
| **API** | Как программы "общаются" друг с другом. |

## 🎯 Результат Meeting 0

К концу этой встречи ты должен:
- [ ] Понимать свою новую роль: PM + QA + дирижёр.
- [ ] Видеть Claude Code на карте инструментов.
- [ ] **Перестать бояться** кода. В Software 3.0 код — это просто текст, который пишет агент.
