# 🗺️ Roadmap: Vibe Coder / AI Generalist

> **Карта пути от нонкодера до AI-generalist'а.** Показывает, где ты сейчас, что дальше, и что получаешь на выходе каждого уровня.

---

## 🎯 Обзор

**AI Generalist** — full-stack решатель проблем, который решает задачи с помощью AI.

Он обладает:
- Глубоким пониманием AI-моделей и их возможностей
- Знанием, какую модель где и как применять
- Умением собирать решения для сложных задач из готовых блоков

### Для кого этот путь

- **Предприниматель** — запускаешь стартап или бизнес и хочешь двигаться быстрее
- **Фрилансер / консультант** — продаёшь экспертизу, масштабируешь выдачу
- **Хастлер** — ищешь новые способы зарабатывать
- **Специалист в найме** — хочешь расти быстрее и быть незаменимым

В "AI First" мире становление AI-generalist'ом перестаёт быть опцией — это базовая грамотность.

## Level 6: Extending the System

От пользователя Claude Code к архитектору собственной системы.

**Юниты:** Чего не хватает · MCP · Hooks · Skills · Подключи и протестируй

**Концепции:**
- MCP (Model Context Protocol): три примитива — Tools, Resources, Prompts
- Tool Search: экономия 85% токенов на инструментах
- Agent Skills — модульные возможности без расхода токенов
- Hooks — автоматизация событий PreToolUse / PostToolUse / Stop
- Superpowers workflow: Brainstorm → Plan → Execute → Verify

**Инструменты:** MCP-серверы (github, stitch, chrome-devtools, vercel), Skills repo, hooks config

**Проекты:**
- Подключённый MCP-сервер и реальный use case
- Установленный или созданный свой Skill
- Настроенный хотя бы один Hook под реальный рабочий сценарий

## 🌐 The Bigger Picture

Соединение всех уровней в твою **"armia агентов-клонов"** — 5 клонов, которые берут рутину на себя, освобождая тебя для стратегических решений.

Пять клонов-кандидатов (см. Модуль 1):
1. **Communication Clone** — пишет письма и сообщения в твоём стиле
2. **Meeting Intelligence Clone** — делает summary встреч, извлекает задачи
3. **Video / Content Clone** — генерирует посты, нарезки, подписи
4. **Learning Clone** — учит тебя новому по расписанию
5. **Automation Clone** — рутинные pipeline (URL → summary, данные → отчёт)

Построение одного клона — это применение всех 6 уровней одновременно:
- Level 1 даёт установку — делегировать, а не командовать
- Level 2–3 настраивает среду и формулирует ТЗ
- Level 4 даёт память клону
- Level 5 делает pipeline
- Level 6 расширяет возможности через MCP и Skills

## ⚡ Summary of Superpowers

Освоив все уровни, ты получаешь четыре силы:

- **Power to Think with AI** — формулировать ТЗ, а не команды; думать категориями спецификации
- **Power to Build with AI** — собирать pipeline и инструменты под свои задачи
- **Power to Automate with AI** — освобождать 5–15 часов в неделю от рутины
- **Power to Network with AI** — делиться паттернами с сообществом и усиливать друг друга
