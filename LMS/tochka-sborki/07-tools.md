# Meeting 6: Инструменты расширения (Tools & MCP) 🔌

> **Цель встречи:** Дать агенту «суперспособности». Мы научим Claude Code выходить в интернет, работать с твоим календарем, почтой и базами данных через протокол MCP.

---

## ⚡ Активация: Агент в клетке

Открой Claude Code и попроси его сделать что-то, что требует выхода во «внешний мир». Например:
> *«Узнай текущую погоду в моем городе»* или *«Найди последние новости про Anthropic за сегодня»*.

**Что он ответит?** Без специальных инструментов он скажет, что его знания ограничены моментом обучения и у него нет доступа к интернету.

---

## 🔄 Рефлексия: Ограниченный интеллект

Представь гениального ученого, запертого в комнате без окон, телефона и интернета.
- Насколько он полезен для решения текущих проблем?
- Что изменится, если дать ему компьютер с доступом ко всем API мира?

**Главный инсайт:** Инструменты (Tools) — это не просто «добавка», это способ превратить чат-бота в **действующего агента**. MCP (Model Context Protocol) — это стандартный разъем, через который мы подключаем эти способности.

---

## 🧠 Концепция: Экосистема расширений

Мы можем расширять Claude Code на трех уровнях:

1.  **MCP-серверы** — Стандартные «разъемы» для внешних сервисов (GitHub, Slack, Google Drive, PostgreSQL).
2.  **Skills (Навыки)** — Твои личные скрипты и инструкции для сложных задач (например, «сделай PDF из этого Markdown»).
3.  **Hooks (Хуки)** — Автоматические действия. Например: «после каждого изменения кода запускай тесты».

### 🤖 Таксономия агентов: на каком уровне ты?

| Уровень | Тип | Что умеет | Примеры |
| :---: | :--- | :--- | :--- |
| 1 | **Responder** | Просто отвечает на вопросы | ChatGPT / Claude.ai |
| 2 | **Tool-User** | Использует файлы, поиск, калькулятор | Claude Code (базовый) |
| 3 | **MCP Agent** | Работает с твоими API, БД, GitHub | **Claude Code + MCP** |
| 4 | **Multi-Agent** | Координирует работу других агентов | Superpowers / CrewAI |

---

## ⚙️ Эксперимент: Подключаем «Глаза» (Веб-поиск)

Давай дадим агенту возможность искать информацию в интернете.

1.  Установи MCP-сервер для поиска (например, Brave Search):
    ```bash
    claude mcp add brave-search -- npx @modelcontextprotocol/server-brave-search
    ```
2.  Добавь API-ключ (Brave Search дает бесплатный лимит).
3.  Попроси Claude: `Найди последние новости про Claude Code`.

**Что изменилось?** Теперь агент не галлюцинирует, а дает ссылки на реальные события.
