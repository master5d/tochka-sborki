## What you need for agentic coding

Think of it as **unboxing**. You've already picked your box — Mac or Windows (the 🍎/🪟 toggle at the top of the page). This unit opens it: we lay out the toolset and check each piece — what's already included, and what to install.

The minimal stack for Claude Code:

| Tool | Why | Check |
|------------|-------|---------|
| **Node.js 18+** | Runs JavaScript/TypeScript, required for the Claude Code CLI | `node --version` |
| **Git** | Code versioning, commits, rolling back changes | `git --version` |
| **Terminal** | Where everything happens | Working? |
| **Claude Code** | The main agent — we install it in the next unit | — |

### How to check your environment in one command

```bash
node --version && git --version && echo "Base environment ready ✓"
```

If there's an error — whichever tool isn't found is the one you need to install.

### What to download if not installed

- **Node.js:** nodejs.org → LTS button (Long Term Support)
- **Git:** git-scm.com → download for your OS
- **Warp (recommended terminal):** warp.dev — a modern terminal with AI hints

> 💡 Warp is optional. VS Code's built-in terminal or your system terminal will also work.

</Phase>
