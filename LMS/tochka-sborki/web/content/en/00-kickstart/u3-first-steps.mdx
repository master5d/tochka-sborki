<Phase type="concept">

## The path from M0 to M6

```
M0 (now): Understood what this is and why
    ↓
M1: Understood how to think about working with AI
    ↓
M2: Set up the tools
    ↓
M3: Can write effective prompts
    ↓
M4: Can manage context
    ↓
M5: Built your first pipeline
    ↓
M6: Plugged in MCP, Hooks, Skills
    ↓
Result: Your first AI clone is running
```

Each Meeting — one concrete skill, one concrete result.

## Vibe coding: what it actually means

**Old way:** write code → run → error → fix → repeat

**Vibe coding:** describe the task → agent writes code → you check → tweak → repeat

You stop being the doer — you become the director.
