## Final architecture: an AI clone for a real task

You can assemble a full specialized agent from:

```
CLAUDE.md          ← context: who you are and what you do
+ MCP server       ← access to needed tools (GitHub, Notion, browser)
+ Skill            ← reusable scenario for your task
+ Hook             ← automation after completion
```

### Example: Meeting Intelligence clone

```
CLAUDE.md: "You are a meeting assistant. Priority: structured outputs"
MCP: Google Calendar (to fetch meetings)
Skill: /meeting-summary (transcript → decisions + tasks)
Hook: PostToolUse → send summary to Slack
```

### Example: Code Review clone

```
CLAUDE.md: "You are a senior developer. Focus: security and readability"
MCP: GitHub (sees PRs and files)
Skill: /code-review (analyze diff → find issues)
Hook: Stop → auto-add a comment to the PR
```
