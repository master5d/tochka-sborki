## Финальная архитектура: AI-клон для реальной задачи

Ты можете собрать полноценного специализированного агента из:

```
CLAUDE.md          ← контекст кто ты и что делаешь
+ MCP-сервер       ← доступ к нужным инструментам (GitHub, Notion, браузер)
+ Skill            ← переиспользуемый сценарий для твоей задачи
+ Hook             ← автоматизация после завершения
```

### Пример: Meeting Intelligence клон

```
CLAUDE.md: «Ты — ассистент по встречам. Приоритет: структурированные результаты»
MCP: Google Calendar (для получения встреч)
Skill: /meeting-summary (суммирует транскрипт → решения + задачи)
Hook: PostToolUse → отправить summary в Slack
```

### Пример: Code Review клон

```
CLAUDE.md: «Ты — senior developer. Фокус: безопасность и читаемость»
MCP: GitHub (видит PR и files)
Skill: /code-review (анализирует diff → находит проблемы)
Hook: Stop → добавить комментарий к PR автоматически
```
