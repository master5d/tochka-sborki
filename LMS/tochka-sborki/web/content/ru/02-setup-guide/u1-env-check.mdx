<Phase type="activation">

Твой компьютер прямо сейчас — это **базовый лагерь перед восхождением**. Есть снаряжение, которое уже лежит в рюкзаке; есть то, чего не хватает.

Не представляй это как «установленный софт». Представь, как альпинист у палатки перебирает карабины, верёвки, кошки — молча проверяет каждый предмет: есть / нет / нужно докупить.

Прокрути в голове этот образ: ты стоишь у рюкзака, достаёшь вещи одну за одной. Что там уже есть? Что ты ещё не видел? **Что может подвести на маршруте?**

</Phase>

<Phase type="reflection">

Вспомни, как ты оказывался в незнакомом месте — и первым делом оглядывался: что здесь есть, куда выходы, что непонятно.

Не думай об этом как «я разобрался с инструментами». Представь это как **раскопки в незнакомом слое грунта**: лопата входит — и ты не знаешь, наткнёшься ли на твёрдый фундамент или на пустоту.

Прокрути в голове этот слой за слоем — **что ты уже нашёл, а что ещё скрыто?** На стыке «незнакомая земля» и «то, что я уже знаю» проступает карта: пробелы становятся видны раньше, чем ты их ощутишь на маршруте.

</Phase>

<Phase type="concept">

## Что нужно для агентного кодирования

Думай об этом как о **распаковке**. Ты уже выбрал свою коробку — Mac или Windows (переключатель 🍎/🪟 вверху страницы). Этот юнит её открывает: достаём набор инструментов и проверяем каждый — что уже в комплекте, а что доустановить.

Минимальный стек для работы с Claude Code:

| Инструмент | Зачем | Проверь |
|------------|-------|---------|
| **Node.js 18+** | Запускает JavaScript/TypeScript, нужен для Claude Code CLI | `node --version` |
| **Git** | Версионирование кода, коммиты, откат изменений | `git --version` |
| **Терминал** | Место, где всё происходит | Работает? |
| **Claude Code** | Главный агент — устанавливаем в следующем unit'е | — |

### Как проверить окружение одной командой

```bash
node --version && git --version && echo "Базовое окружение готово ✓"
```

Если ошибка — какой инструмент не найден, тот и нужно установить.

### Что скачать если не установлено

- **Node.js:** nodejs.org → кнопка LTS (Long Term Support)
- **Git:** git-scm.com → скачать для своей ОС
- **Warp (рекомендуемый терминал):** warp.dev — современный терминал с AI-подсказками

> 💡 Warp — опционально. Встроенный терминал VS Code или системный тоже подойдёт.

</Phase>

Если что-то не установлено — скачай с официального сайта и установи. После установки проверь снова.
