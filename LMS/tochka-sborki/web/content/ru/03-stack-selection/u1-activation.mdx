<Phase type="activation">

Вспомни свою AI-среду прямо сейчас — бюджет, железо, страну, корпоративные ограничения.

Не описывай это как «технический выбор инструмента». Представь свою ситуацию как **путь на гору: есть канатка и есть пешая тропа** — и только ты знаешь, что у тебя в рюкзаке, сколько времени до темноты и нужна ли тебе вершина сегодня или просто панорама.

Прокрути в голове: какой маршрут ощущается как твой? Где для тебя «канатка» (быстро, платно, удобно) — и где «тропа» (медленнее, автономно, твоё)?

Удержи это ощущение — и ответь на 5 вопросов ниже мысленно:

1. **Бюджет:** сколько ты реально готов платить за AI-инструменты в месяц?
   - $0 / до $5 / до $20 / до $50 / без ограничений

2. **Доступ:** где ты находишься?
   - Mainland China без VPN / страна с GDPR / США / другая

3. **Железо:** что у тебя есть?
   - Только ноут без GPU / Mac M-series / PC с дискретной NVIDIA / удалённый сервер

4. **Приватность:** насколько важно чтобы код не покидал твою машину?
   - Не важно / желательно / критично (NDA, медицина, инфра)

5. **Готовность настраивать:** сколько часов на первичный сетап ты готов потратить?
   - 15 мин / 1 час / целый вечер / несколько дней

</Phase>

<Phase type="reflection">

Твои 5 ответов — это не таблица предпочтений. Это **дегустационная карта** 🍷.

Представь, что сомелье берёт бокал не по названию этикетки, а по запаху, послевкусию, кислотности — по тому, что есть в реальном бокале. Точно так же разгляди, что реально в твоём стакане: деньги, железо, страна, параноя по приватности.

На стыке «мои ответы» и «карта стеков» — уже проступает одна-две чётких строки:

| Если у тебя… | …то скорее всего тебе нужен |
|---|---|
| Бюджет $20+, никаких ограничений, минимум настройки | **Claude Code** — paid но удобный |
| Бюджет $0, есть железо, важна автономность | **Sovereign** (по SOVERN) — Hermes + локальные модели |
| Бюджет $0–5, нет железа, доступ есть | **Cloud-OSS** — Cline + free-tier клауды |
| В Китае, без VPN | **Behind-GFW** — Cerebras + Gemini Flash + local |

Не «правильного» ответа нет — есть стек, который подходит **именно твоему бокалу**.

</Phase>

<Phase type="concept">

## Почему нельзя «просто Claude Code»

Курс изначально написан под Claude Code — это сильный, удобный инструмент. Но он:

- **Платный** — минимум ~$20/мес на Sonnet
- **Требует доступа** к `claude.ai` и API Anthropic. В Китае без VPN — недоступен
- **Vendor-lock-in** — твои промпты, MCP-конфиги, hooks привязаны к экосистеме Anthropic
- **Telemetry** — каждый запрос идёт в Anthropic. Не все corporate / госы это разрешают

Для большинства студентов это OK. Но если хотя бы один из четырёх пунктов выше — стоп, то Claude Code тебе не подходит. И ты не «хуже» — просто **agentic AI больше не моно-инструмент**, есть открытые альтернативы.

## Что такое «стек»

В контексте этого курса **стек** = комбинация трёх слоёв:

```
┌─────────────────────────────────────────────┐
│  Frontend / Agent UI                        │  ← Claude Code, Cline, Aider, Hermes
├─────────────────────────────────────────────┤
│  Routing / Gateway                          │  ← LiteLLM, OpenRouter, прямой API
├─────────────────────────────────────────────┤
│  Model provider (где живёт интеллект)       │  ← Anthropic, Cerebras, Groq, local GGUF
└─────────────────────────────────────────────┘
```

Каждый слой можно менять независимо. Например:
- **Claude Code → LiteLLM → Cerebras Qwen-3-235B** — Claude Code UI с открытой моделью
- **Aider → прямой OpenRouter → DeepSeek-Coder** — классический OSS workflow
- **Hermes → LiteLLM с fallback массивом** — SOVERN-pattern

Этот модуль научит тебя **думать про стек целиком**, а не «знать одну тулзу».
