# Meeting 2: Базовый сетап (Setup Guide) 🛠️

> **Цель встречи:** Превратить твой компьютер в лабораторию для агентного кодирования. Мы установим «руки и глаза» для Claude Code.

---

## ⚡ Быстрая установка (одной строкой)

Нетехнический путь — одна команда ставит весь базовый стек (Node.js + Git + Claude Code) и проверяет результат:

- **Mac / Linux:** `curl -fsSL https://ai.mamaev.coach/install.sh | bash`
- **Windows:** `irm https://ai.mamaev.coach/install.ps1 | iex`

После установки открой **новый** терминал и запусти `claude`. Если установщик выдал ошибку — ниже пошаговый ручной путь.

---

## ⚡ Активация: Проверка готовности

Прежде чем строить ракету, проверим, есть ли у нас молоток и отвёртка. Открой терминал (в Windows это PowerShell или CMD, в macOS — Terminal) и введи:

```bash
node --version
git --version
```

**Что ты видишь?** Номер версии или «команда не найдена»? Если второе — не переживай, сейчас всё исправим.

---

## 🔄 Рефлексия: Боль установки

Вспомни последний раз, когда тебе нужно было настроить новый инструмент или окружение.
- Сколько времени ушло на «dependency hell» (ад зависимостей)?
- Было ли ощущение, что ты делаешь магические действия, не понимая зачем?

В Software 3.0 сетап — это **единоразовое действие**. Мы настраиваем фундамент, чтобы потом агент сам заботился о деталях внутри проектов.

---

## 🧠 Концепция: Минимальный стек

Для работы с агентами нам нужны 4 уровня инструментов:

1.  **Среда выполнения (Node.js)** — «двигатель», на котором работает Claude Code.
2.  **Версионирование (Git)** — твоя «машина времени» и страховка от ошибок.
3.  **Интерфейс (Warp / Terminal)** — место, где ты общаешься с агентом.
4.  **Сам агент (Claude Code)** — твой главный напарник.

---

## ⚙️ Эксперимент: Пошаговая установка

### Шаг 1: Базовые инструменты

Если команды из «Активации» выдали ошибку — скачай и установи:
- **Node.js**: [nodejs.org](https://nodejs.org) (бери версию **LTS**).
- **Git**: [git-scm.com](https://git-scm.com).

### Шаг 2: Установка Claude Code

Это твой основной инструмент. Устанавливается одной командой в терминале:

```bash
npm install -g @anthropic-ai/claude-code
```

**Проверь:** `claude-code --version`. Если видишь цифры — магия началась.

### Шаг 3: Настройка «машины времени» (Git)

Чтобы агент мог сохранять твой прогресс, представься системе:

```bash
git config --global user.name "Твоё Имя"
git config --global user.email "your@email.com"
```

## 📂 Твоя рабочая область

Создадим структуру, в которой агент будет чувствовать себя как дома:

```bash
mkdir -p ~/ai-projects/experiments
cd ~/ai-projects
```

---

## 📄 Главный файл: CLAUDE.md

Это «инструкция для входа» в твой проект. Когда Claude Code заходит в папку, он первым делом ищет этот файл, чтобы понять, кто ты и что мы строим.

**Как создать быстро:**
1.  Зайди в папку проекта.
2.  Напиши `/init` внутри сессии `claude-code`.

Создай свою первую рабочую папку:

```bash

cd hello-world
git init

```
