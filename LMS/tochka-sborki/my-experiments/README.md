# my-experiments 📝

Это твоя песочница для экспериментов. Здесь ты сохраняешь результаты упражнений и пробуешь новые идеи.

## Структура

### Результаты упражнений из курса

```
exercise-1-delegation.md         # Meeting 3: Упр 1 - переписать команды
exercise-2-iterations.md          # Meeting 3: Упр 2 - 3 итерации промпта
exercise-3-context-files.md       # Meeting 4: Упр 3 - CLAUDE.md, TODO.md, AGENTS.md
exercise-4-explore.md             # Meeting 4: Упр 4 - результаты /explore
exercise-5-meta-prompt.md         # Meeting 3: Упр 5 - мета-промптинг
exercise-6-parallel-agents.md     # Meeting 4: Упр 6 - параллельные агенты
exercise-7-my-template.md         # Meeting 3: Упр 7 - твой шаблон
exercise-8-working-style.md       # Meeting 3: Упр 8 - твой стиль работы
```

### Твои собственные эксперименты

Создавай новые файлы с именем вида `experiment-[название].md`:

```
experiment-new-prompt-style.md    # Новый стиль написания промптов
experiment-context-rot.md          # Попытка избежать Context Rot
experiment-delegation-matrix.md   # Своя матрица делегирования
```

## Как использовать

1. **После упражнения:** Сохрани результат с названием из списка выше
2. **Личные эксперименты:** Придумай свой эксперимент и документируй результат
3. **Анализ:** Время от времени пересматривай, что сработало и что нет

## Структура файла

```markdown
# [Название эксперимента]

## Цель
Что я хотел попробовать?

## Ход работы
Как я это делал?

## Результат
Что получилось?

## Вывод
Что я узнал? Сработало ли?

## Дата
Когда это было?
```

## Примеры экспериментов

### Пример 1: Новый стиль промпта
```markdown
# Experiment: Метод 5W1H в промптах

## Цель
Попробовать писать промпты используя Who, What, When, Where, Why, How

## Ход работы
Переписал свой обычный промпт в стиле 5W1H...

## Результат
Агент дал результат на 30% быстрее...

## Вывод
5W1H помогает структурировать - буду использовать для сложных задач

## Дата
2026-03-25
```

### Пример 2: Снижение Context Rot
```markdown
# Experiment: Минимальный контекст

## Цель
Пробую работать с минимальным контекстом - только CLAUDE.md, без дополнительных файлов

## Результат
Качество упало, агент спрашивает слишком много...

## Вывод
Контекст должен быть достаточным, но не переполненным - нужна золотая середина

## Дата
2026-03-26
```
