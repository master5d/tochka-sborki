# DESIGN.md: MAMA_MENTOR (High-End Agentic Engineering)

## 1. Ethos
We don't sell "consulting." We ship **Infrastructural Intelligence**. The design must feel like a **Mission Control Center** — high-density, authoritative, and technically superior. No "fluff," no generic AI gradients.

## 2. Visual DNA (Archetype: Linear + Stripe + Cyberpunk)
- **Palette (Cyber Blue):**
  - Primary: `#00d1ff` (Cyber Blue)
  - Background: `#050508` (Deep Abyss)
  - Surface: `#0a0a14` (Steel Shell)
  - Border: `rgba(0, 209, 255, 0.15)`
- **Typography:**
  - Headers: `Unbounded` (Weight: 900) - Tight tracking (-0.05em).
  - Body: `Geist Sans` (Weight: 400) - Optimal line-height (1.62).
  - Data: `Geist Mono` (Weight: 700) - For status points and metrics.
- **Atmosphere:**
  - 1px borders only.
  - Subtle scanline overlay on backgrounds.
  - Backdrop blur: 20px (Elite Glassmorphism).
  - High data density (OCD-level alignment).

## 3. Component Standards
- **Command Cards:** No shadows. Use 1px borders and inner glows (`box-shadow: inset 0 0 10px rgba(0, 209, 255, 0.05)`).
- **Status Pills:** Mono font, uppercase, 0.15em letter-spacing.
- **Grids:** Use a 12-column Swiss grid with 1.5rem gutters.

## 4. Anti-Patterns (The "Slop" Filter)
- ❌ **No Inter font** for display headers.
- ❌ **No generic cards.** Use "Dashboard Panels."
- ❌ **No emojis.** Use custom SVG icons or mono-symbols.
- ❌ **No purple/pink gradients.** Stick to Cyber Blue and Steel.

## 5. Layout (The Dashboard)
- **Hero:** Asymmetrical. Left-aligned text, right-aligned "Live Feed" or "Observability Graph."
- **Services:** Structured as "System Modules" with specific SVC-SKILL codes.
- **Process:** A linear "Compilation Chain."
