Глобальный файл памяти. Загружается для всех проектов. Личные настройки и предпочтения.

Системный каталог конфигурации Claude Code. Для enterprise-настроек.

Правила проекта в отдельных файлах. Каждый .md файл — отдельное правило. Поддерживает глобы для условной загрузки.

Пример

📋 Процессы (AGENCY)

5
Workflow: Plan → Normal → AutoAGENCY Lab

Золотой паттерн работы: 1) Plan — разведка и планирование, 2) Normal — реализация с контролем, 3) Auto — рутинные операции без подтверждений.

Пример
Plan: 'проанализируй структуру' → Normal: 'реализуй план' → Auto: 'обнови импорты'

Постепенное повышение автономии по мере уверенности в задаче

Частые ошибки новичковAGENCY Lab

Типичные ошибки: 1) Не использовать /compact, 2) Слишком длинный CLAUDE.md, 3) Не проверять /context, 4) Всегда в одном режиме, 5) Не использовать Plan для разведки.

CLAUDE.md: best practicesAGENCY Lab

Держите CLAUDE.md кратким: ключевые технологии, правила, ограничения. Объёмные описания → в .claude/rules/*.md. Обновляйте регулярно.

Пример
# Проект
- Stack: Next.js 15, TS, Tailwind
- Tests: vitest, test before commit
- Style: functional, no classes
- NEVER: commit .env files

Краткий, но информативный CLAUDE.md

Когда какой субагентAGENCY Lab

Explore — для понимания кодовой базы. Plan — для сложных задач с планом. General — для подзадач. Bash — для скриптов и CLI-операций.

🔌 MCP-серверы

6
--transport http

Подключить MCP-сервер через HTTP-транспорт (streamable HTTP). Рекомендуемый способ для удалённых серверов.

Пример

Подключает удалённый MCP-сервер через HTTP

Подключить MCP-сервер через stdio (стандартный ввод/вывод). Для локальных серверов.

--transport sse

Подключить MCP-сервер через SSE (Server-Sent Events). Устаревший вариант HTTP.

Scopes: local / project / user

MCP-серверы имеют три области видимости: local (.mcp.json), project (.claude/settings.json), user (~/.claude/settings.json).

Показать все подключённые MCP-серверы и их статус.

claude mcp serve

Запустить Claude Code как MCP-сервер. Позволяет другим приложениям использовать Claude Code.

🎯 Контекст (AGENCY)

5
Context rot (деградация контекста)AGENCY Lab

Явление ухудшения качества ответов при переполнении контекстного окна. Симптомы: повторения, забывание инструкций, галлюцинации.

Lost in the MiddleAGENCY Lab

Эффект: информация в середине длинного контекста хуже усваивается LLM. Начало и конец — лучше.

Дерево решений: /clear vs /compactAGENCY Lab

Когда /clear: новая задача, Claude Code путается, контекст >80%. Когда /compact: та же задача, нужна история решений, контекст 50-80%.

Пример
Контекст 75%, та же задача → /compact
Контекст 85%, новая задача → /clear

Простое правило выбора между сжатием и полной очисткой

Здоровье контекста (/context)AGENCY Lab
