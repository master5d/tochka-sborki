Сайдбар читает этот объект при монтировании и показывает ✓/→/○ для каждого unit'а
05 / Новые компоненты
UnitWizard Оборачивает Phase-компоненты. Управляет currentStep (useState). Показывает прогресс-бар и кнопки Далее/Готово.
Phase type: 'activation'|'reflection'|'concept'|'practice'. Рендерит заголовок фазы + children. Видима только если currentStep совпадает.
UnitSidebar Расширяет текущий Sidebar. Читает unit_progress из localStorage. Показывает unit'ы как вложенный список под meeting'ом.
MeetingIndex Страница /lessons/[meeting]/. Сервер-компонент, читает _meta.json. Редиректит на первый незавершённый unit (client-side после чтения localStorage).
