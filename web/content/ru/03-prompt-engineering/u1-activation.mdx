<Phase type="activation">

Вспомни промпт, который ты когда-либо давал AI — и получил разочаровывающий или бесполезный ответ.

**Что именно ты написал?** Что AI ответил вместо того, что ты хотел?

</Phase>

<Phase type="reflection">

Посмотри на свой промпт со стороны — глазами человека, который его читает впервые:

- Какая роль у того, кто должен отвечать?
- Что конкретно нужно получить на выходе?
- Какие ограничения или критерии качества были указаны?

Скорее всего — ничего из этого не было. **AI делал предположения за тебя.**

</Phase>

<Phase type="concept">

## Почему плохие промпты дают плохие результаты

LLM — это машина предсказания. Она продолжает твой текст так, как написал бы типичный человек в этом контексте.

Когда ты пишешь `«Напиши статью про маркетинг»` — модель угадывает:
- Для кого? (случайной аудитории)
- Какого размера? (обычного)
- В каком стиле? (нейтральном)
- С каким фокусом? (общим)

**Формула плохого промпта:**
```
Задача без контекста = предположения без критериев = лотерея
```

**Формула хорошего промпта:**
```
Роль + Задача + Входные данные + Ожидаемый результат + Ограничения = воспроизводимый результат
```

| Плохой промпт | Хороший промпт |
|---------------|----------------|
| 2–3 слова | 5–15 предложений |
| 5–7 итераций | 1–2 итерации |
| 60% попадание | 95% попадание |
| Каждый раз разный | Воспроизводимый |

</Phase>
