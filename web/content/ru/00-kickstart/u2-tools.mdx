Что бы ты хотел попросить AI сделать для тебя регулярно? Не «что он умеет», а что было бы полезно лично тебе каждый день или каждую неделю?

<Phase type="reflection">

Большинство людей используют AI как «умный поиск» — задают вопрос, получают ответ, закрывают.

Следующий уровень — AI как **рабочий инструмент**: тебе не нужно объяснять заново кто ты, что за проект, какие правила. Он уже знает.

Для этого нужен Claude Code, а не просто чат.

</Phase>

<Phase type="concept">

## Три уровня работы с AI

### Уровень 1: Чат (сейчас)
```
Ты → Вопрос → Claude → Ответ
```
Каждый раз заново объясняешь контекст. Подходит для разовых задач.

### Уровень 2: Claude Code (цель этого курса)
```
Ты → Задача в проекте → Claude знает контекст из CLAUDE.md → Результат
```
Агент видит файлы, запускает команды, помнит проект.

### Уровень 3: Автоматизация (Meeting 5–6)
```
Trigger → Claude делает задачу → Результат → Уведомление тебе
```
Агент работает без твоего участия.

## Что установить для начала

1. **claude.ai** (браузер/телефон) — чат, уже работает
2. **Node.js** — нужен для Claude Code CLI
3. **Claude Code** — `npm install -g @anthropic-ai/claude-code`

Детальная установка — в Meeting 2.

> 💡 Это Kickstart, не установочная встреча. Просто понимай карту. Всё настроим в M2.

</Phase>
