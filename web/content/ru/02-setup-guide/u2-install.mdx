## Установка Claude Code

Claude Code — это CLI (command-line interface), агент который работает прямо в терминале.

### Установка

```bash
npm install -g @anthropic-ai/claude-code
```

### Проверка

```bash
claude --version
claude --help
```

### Первый запуск

```bash
cd ~/Documents
claude
```

Claude Code откроет интерактивную сессию. Первый раз попросит войти через браузер.

### Git: базовая конфигурация

```bash
git config --global user.name "Твоё имя"
git config --global user.email "твоя@почта.com"

# Проверь:
git config --global user.name
```

### Полезные настройки терминала

```bash
# Добавь в ~/.bashrc или ~/.zshrc — полезные алиасы
alias ll='ls -lah'
alias gst='git status'
alias gp='git push'
```

> 💡 После добавления алиасов: `source ~/.bashrc` (или перезапусти терминал).
