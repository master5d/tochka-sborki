<Phase type="activation">

Представь: Claude Code — посольство твоей страны в чужом городе. Без посольства ты не можешь официально говорить с местной властью — есть барьер протокола.

Не представляй это как «технический вопрос». Представь молчаливый кордон между государствами: без посла и переводчика стороны смотрят друг на друга и не могут обменяться ни словом. Какой внешний «сосед» сейчас молча стоит у твоего кордона — GitHub, Notion, браузер, Telegram?

Удержи его в голове: **что первым нарушит молчание?**

</Phase>

<Phase type="reflection">

MCP — это общий дипломатический язык: посольство уже знает, как говорить с любой страной через единый протокол.

Не думай об этом как о «подключении инструментов». Представь телеграфный кабель между двумя станциями: радиооператор с обеих сторон знает азбуку Морзе — и не важно, кто там на другом конце, сигнал пройдёт. Один протокол вместо тысячи двусторонних договоров.

Прокрути в голове: если MCP — это общая азбука, какую первую «телеграмму» ты бы отправил?

</Phase>

## MCP: три примитива

| Примитив | Управляется | Что это |
|----------|-------------|---------|
| **Tools** | Моделью | Функции, которые агент вызывает: API-запросы, запись файлов |
| **Resources** | Приложением | Источники данных (как GET-запросы): файлы, записи БД |
| **Prompts** | Пользователем | Шаблоны для оптимального использования |

## Как подключить MCP-сервер

В файл `.claude/settings.json` добавь:

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "твой-токен"
      }
    }
  }
}
```

## Популярные MCP-серверы

| Сервер | Что даёт |
|--------|---------|
| `@modelcontextprotocol/server-github` | Issues, PR, commits, файлы репозитория |
| `@modelcontextprotocol/server-filesystem` | Расширенный доступ к файловой системе |
| `@modelcontextprotocol/server-brave-search` | Веб-поиск через Brave |
| Notion MCP | Чтение и запись страниц Notion |
| Firecrawl | Скрапинг веб-страниц |

## Частые ошибки (~60% проблем)

- `Server transport closed unexpectedly` — синтаксическая ошибка в JSON-конфиге
- Конфликты портов — два сервера пытаются слушать один порт
- Нет переменной окружения — токен не передан правильно

Решение: проверь JSON-синтаксис и токены прежде чем искать глубже.
