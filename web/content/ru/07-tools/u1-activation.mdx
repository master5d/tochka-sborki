> 💡 **MCP — открытый стандарт.** Этот модуль показывает MCP/Hooks/Skills через Claude Code. **MCP-серверы поддерживаются также** в Cline, Continue.dev, Goose, Cursor. Hooks и Skills — Claude-Code-specific, но эквиваленты есть в каждом стеке (см. модуль 03).

<Phase type="activation">

Вспомни момент, когда Claude Code остановился на полпути — нужный сервис был рядом, но агент его не видел.

Не описывай это как «баг» или «ограничение». Представь мастера у верстака: пояс инструментов застёгнут, но нужный ключ не вложен — рука тянется в пустой кармашек, работа встаёт. Прокрути в голове: что именно было в том пустом кармашке?

**Какой инструмент ты бы вложил туда первым?**

</Phase>

<Phase type="reflection">

Claude Code «из коробки» — это штурман с точными картами звёздного неба, но без секстанта: знает, куда плыть, только не может взять высоту здесь и сейчас.

Не думай об этом как об «установке плагинов». Представь, что штурман наконец получает инструмент: одно измерение — и позиция известна. Три механизма дают агенту эти инструменты: **MCP** (подключение сервисов), **Hooks** (автоматизация событий), **Skills** (переиспользуемые сценарии).

Удержи в голове: какое первое «измерение» разблокирует твою работу?

</Phase>

<Phase type="concept">

## Таксономия расширений Claude Code

| Механизм | Что делает | Аналогия |
|----------|-----------|---------|
| **MCP Server** | Даёт агенту новые инструменты (GitHub, Notion, браузер) | Установить приложение |
| **Hook** | Запускает код при событиях (до/после действия агента) | Автоматический триггер |
| **Skill** | Переиспользуемый сценарий, вызываемый через `/skill-name` | Макрос / скрипт |

## Уровни агентности

| Уровень | Тип | Что умеет |
|---------|-----|-----------|
| 1 | Basic Responder | Отвечает на вопросы |
| 2 | Router | Направляет запрос к нужному агенту |
| **3** | **Tool-Calling** | **Использует внешние инструменты — Claude Code с MCP** |
| 4 | Multi-Agent | Несколько агентов работают вместе |
| 5 | Autonomous | Действует без участия человека |

Claude Code с MCP — это уровень 3. С Skills и параллельными агентами — уровень 4.

</Phase>
