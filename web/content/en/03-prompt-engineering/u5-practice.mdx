## Meta-prompting: a prompt that writes prompts

If you don't know where to start — ask Claude to write the prompt for you:

```
You are an expert at writing prompts for language models.

Write a detailed prompt for an assistant that will [task description].

Each of its responses must:
- [criterion 1]
- [criterion 2]
- [criterion 3]

Include all parameters important for quality.
```

### Template for a reusable assistant

For repeating tasks (Claude Project, Gemini Gem, Claude Code Skill):

```
Role:         who this assistant is (expertise, style, boundaries)
Objective:    the main goal — why it exists
Context:      the situation it's used in
Instructions: list of rules it operates by
Notes:        reminders, edge cases, prohibitions
```

### Checklist before you hit send

- [ ] Role is clear (who's answering, with what expertise)?
- [ ] Task is specific (what exactly is needed)?
- [ ] Output is defined (how do you know it's right)?
- [ ] Constraints are clear (what not to do)?
- [ ] No distractors (no extra text)?
- [ ] Important stuff up front (not buried in the middle)?

</Phase>
