## Installing Claude Code

Claude Code is a CLI (command-line interface), an agent that works right in your terminal.

### Install

```bash
npm install -g @anthropic-ai/claude-code
```

### Check

```bash
claude --version
claude --help
```

### First run

```bash
cd ~/Documents
claude
```

Claude Code will open an interactive session. The first time it'll ask you to log in via your browser.

### Git: basic config

```bash
git config --global user.name "Your name"
git config --global user.email "your@email.com"

# Check:
git config --global user.name
```

### Handy terminal settings

```bash
# Add to ~/.bashrc or ~/.zshrc — useful aliases
alias ll='ls -lah'
alias gst='git status'
alias gp='git push'
```

> 💡 After adding aliases: `source ~/.bashrc` (or restart your terminal).
