## Installing Claude Code

Claude Code is a CLI (command-line interface), an agent that works right in your terminal.

### Install

```bash
npm install -g @anthropic-ai/claude-code
```

> 💡 `npm install -g` works the same in bash and PowerShell.

### Check

```bash
claude --version
claude --help
```

### First run

<OsBlock os="mac">

```bash
cd ~/Documents
claude
```

</OsBlock>

<OsBlock os="windows">

```powershell
cd $HOME\Documents
claude
```

</OsBlock>

Claude Code will open an interactive session. The first time it'll ask you to log in via your browser.

### Git: basic config

Same on every system:

```bash
git config --global user.name "Your name"
git config --global user.email "your@email.com"

# Check:
git config --global user.name
```

### Handy terminal settings

<OsBlock os="mac">

```bash
# Add to ~/.bashrc or ~/.zshrc — useful aliases
alias ll='ls -lah'
alias gst='git status'
alias gp='git push'
```

> 💡 After adding aliases: `source ~/.bashrc` (or restart your terminal).

</OsBlock>

<OsBlock os="windows">

PowerShell uses a profile script instead of `.bashrc`. Open the profile:

```powershell
# Create it if it doesn't exist yet
if (!(Test-Path $PROFILE)) { ni $PROFILE -ItemType File -Force }
# Open in editor
notepad $PROFILE
```

Add functions (PowerShell aliases don't take parameters cleanly, so use functions):

```powershell
function gst { git status }
function gp  { git push }
function ll  { Get-ChildItem -Force }
```

> 💡 After saving: `. $PROFILE` (dot-space) or restart PowerShell.

</OsBlock>
