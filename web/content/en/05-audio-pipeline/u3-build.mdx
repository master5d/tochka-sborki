<Phase type="concept">

## Full pipeline: URL → Insights

This is a working prompt you can use right now:

```
You are an analytical agent. Your job is to extract maximum value from content.

URL to analyze: [paste link]

Tasks:
1. Read the page fully
2. Pull out 5–7 actionable ideas (concrete, not generic phrases)
3. Identify 5–7 topics for further research
4. Find 2–3 best quotes or facts

Output format:
## Key ideas
[list with explanations]

## Topics to research
[list with short context]

## Worth remembering
[quotes or facts]

Save the result to a file ideas.md
```

## Extended pipeline: multiple sources

You can analyze several URLs in parallel:

```
Analyze three articles on the topic [topic]:
1. [URL 1]
2. [URL 2]
3. [URL 3]

For each: 3 ideas + 1 best quote.
Then: find what they have in common and where they contradict.
Save to research-[topic].md
```

</Phase>
