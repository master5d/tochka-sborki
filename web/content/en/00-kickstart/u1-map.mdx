<Phase type="concept">

## The map: 4 quadrants of AI tools

| Quadrant | What it does | Examples |
|----------|-----------|---------|
| 🟦 **Zero-code builders** | Prompt → working app in minutes | Lovable, Bolt, Replit |
| 🟪 **AI dev platforms** | Deep work with code and projects | **Claude Code**, Codex |
| 🟨 **Design-to-code** | Design → code | Figma Make, Stitch |
| 🟩 **AI coding assistants** | Helpers inside your IDE | Cursor, Windsurf |

## Where this course sits

**Claude Code** — quadrant 🟪. This isn't just a chat. It's an agent that:
- Sees your file system
- Runs commands in your terminal
- Remembers context through CLAUDE.md
- Extends through plugins and Skills

## Why not zero-code?

Lovable and Bolt are great for simple landing pages. For serious work:
- You don't get control over the code
- You can't integrate with existing projects
- No automation, no agents

**This course is about the second path.** Owning the tool, not a quick start.

</Phase>
