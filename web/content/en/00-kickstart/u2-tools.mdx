<Phase type="reflection">

Most people use AI like a "smart search" — ask a question, get an answer, close the tab.

The next level is AI as a **work tool**: you don't have to re-explain who you are, what the project is, what the rules are. It already knows.

For that you need Claude Code, not just a chat.

</Phase>

<Phase type="concept">

## Three levels of working with AI

### Level 1: Chat (where you are now)
```
You → Question → Claude → Answer
```
Every time you re-explain the context. Fine for one-off tasks.

### Level 2: Claude Code (the goal of this course)
```
You → Task in a project → Claude knows the context from CLAUDE.md → Result
```
The agent sees files, runs commands, remembers the project.

### Level 3: Automation (Meetings 5–6)
```
Trigger → Claude does the task → Result → You get notified
```
The agent works without you in the loop.

## What to install to get started

1. **claude.ai** (browser/phone) — chat, already works
2. **Node.js** — required for the Claude Code CLI
3. **Claude Code** — `npm install -g @anthropic-ai/claude-code`

Detailed install in Meeting 2.

> 💡 This is Kickstart, not a setup session. Just understand the map. We'll wire it all up in M2.

</Phase>
