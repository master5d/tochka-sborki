<Phase type="activation">

Think back to the last time you tried to do something with AI (ChatGPT, Claude, Gemini — doesn't matter).

**What exactly did you ask for?** What did you get back?

</Phase>

<Phase type="reflection">

Look at that experience honestly:

- Did you get what you wanted — on the first try?
- If not — why? What was unclear in your request?
- What would you change if you could write the request from scratch?

Most people land on the same realization here: **the problem isn't the AI, it's how we talk to it.**

</Phase>

<Phase type="concept">

## Three eras of programming

**Software 1.0** — we wrote code by hand, line by line. The computer did exactly what we said.

**Software 2.0** — machines started learning from data. Neural networks. We give examples, not rules.

**Software 3.0** — **agents write code for us.** We direct, delegate, and check the result.

| Before | Now |
|------|-------|
| I write code → machine runs it | I write a prompt → agent writes code → I review |
| I solve EVERY problem myself | I choose: what to solve, what to delegate |
| Tools help me write | Tools write for me |

> 💡 An AI agent isn't a calculator. It's a junior teammate. It needs context, a goal, and a bit of trust.

</Phase>
