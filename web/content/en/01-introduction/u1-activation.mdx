<Phase type="activation">

Picture the last time you asked something from AI — and got back something close, but not quite right.

Don't analyze it as a "bad prompt." **Instead, picture yourself in a restaurant where the menu is half in a language you only partially know.** You guess at words, point at a line, the waiter nods — and brings something adjacent to what you meant. No one's fault. Just an untranslated word sitting between you.

Run that moment in your head: where exactly did the untranslated word sit between your thought and the AI's answer? Catch it — hold it there.

</Phase>

<Phase type="reflection">

Now shift the frame. Not the restaurant — **a map printed in someone else's handwriting.** You know where you want to go, but half the labels are smudged. The AI navigates by the map you handed it.

Notice in your head: which landmarks on your map were sharp — and which were missing? Where did the AI invent a route instead of reading one?

**Most people catch the same thing here: the problem isn't the navigator — it's the map we hand over.**

</Phase>

<Phase type="concept">

## Three eras of programming

**Software 1.0** — we wrote code by hand, line by line. The computer did exactly what we said.

**Software 2.0** — machines started learning from data. Neural networks. We give examples, not rules.

**Software 3.0** — **agents write code for us.** We direct, delegate, and check the result.

| Before | Now |
|------|-------|
| I write code → machine runs it | I write a prompt → agent writes code → I review |
| I solve EVERY problem myself | I choose: what to solve, what to delegate |
| Tools help me write | Tools write for me |

> 💡 An AI agent isn't a calculator. It's a junior teammate. It needs context, a goal, and a bit of trust.

</Phase>
