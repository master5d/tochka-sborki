## 💡 Ключевые идеи в одном месте

### Четыре сдвига
1. От команд → к делегированию
2. От результата → к эксперименту
3. От действий → к спецификациям
4. От "всё сам" → к управлению вниманием

### Три файла памяти
- **CLAUDE.md** - контекст проекта
- **TODO.md** - текущие задачи
- **AGENTS.md** - инструкции для агента

### Матрица делегирования
**Делегировать:** рутина, черновик, поиск, известные критерии
**Решать сам:** стратегия, оценка, неопределённость, риски

### Структура промпта
```
## Роль
## Задача
## Input
## Output
## Constraints
```
